# jelly
Cellular Automata Experiments

## Cool commands


Nice drawings:
```
python -m jelly.life test.mp4 500 720 1280 1 blues --b 4 --b 6 --b 7 --b 8 --s 3 --s 5 --s 6 --s 7 --s 8
```

Mazes:
```
python -m jelly.life test.mp4 500 100 100 10 blues --b 3 --s 0 --s 1 --s 2 --s 3 --s 4
```


Nice blobs:
```
python -m jelly.life test.mp4 500 720 1280 1 blues --b 3 --b 5 --b 6 --b 7 --b 8 --s 5 --s 6 --s 7 --s 8
```
